# Return of Serve Secret (Third Shot Sports)

**Channel:** Third Shot Sports
**URL:** https://www.youtube.com/watch?v=DDM_aIkKDo4
**Duration:** 1:54 (short) | **Uploaded:** 2024-01-16

## Key Takeaway

Stand well behind the baseline when returning serve. Pros like Anna Leigh Waters, Ben Johns, JW Johnson position a meter or more behind the baseline. This gives more time to set up the stroke and then move forward through the ball to approach the kitchen. Standing close to the line = rushed, off-balance returns and getting stuck in no-man's-land.
